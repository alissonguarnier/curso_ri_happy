Agrupamento de seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separa os seletores utiizando uma vírgula (,)

Assim, não precisamos ficar copiando e colando os mesmos código para todos, nós podemos fazer um só bloco de css e reutilizas as mesmas regras!
