Seletores CSS:

  • Seletor de tipos: Busca elementos por uma tag HTML especifica
  • Seletor por id (#): Busca elementos através do atributo "id"
  • Seletor por Classe (.): Busca elementos através do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo([atrib]): Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com valores específicos
